# svirsh

[![CircleCI](https://circleci.com/gh/youyo/svirsh.svg?style=svg)](https://circleci.com/gh/youyo/svirsh)

## Description

Simple virt-install command.

## Usage

```
$ svirsh -config config.toml
```

## Install

To install, use `go get`:

```bash
$ go get -d github.com/youyo/svirsh
```
